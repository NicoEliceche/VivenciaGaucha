# Experiencia Gaucha Magdalena - Web App

Proyecto ReactJS profesional y moderno siguiendo la arquitectura **feature-based** y el skill **project-conventions**.

## 🚀 Cómo empezar

Este proyecto utiliza **Vite** para un desarrollo rápido.

### Requisitos
- Node.js (v18 o superior recomendado)
- npm o yarn

### Instalación
1. Clona el repositorio o descarga los archivos.
2. Abre una terminal en la carpeta del proyecto.
3. Ejecuta:
   ```bash
   npm install
   ```

### Desarrollo
Para ver el proyecto en tu navegador con recarga automática:
```bash
npm run dev
```

## 🏗️ Arquitectura

El proyecto sigue las siguientes convenciones de diseño y arquitectura:

- **Mobile First**: Diseño optimizado para dispositivos móviles y adaptativo para escritorio.
- **Feature-Based**: El código está organizado por funcionalidades en `src/features/`.
- **Styled Components**: Separación de estilos en archivos `Styled.ts`.
- **Design System**: Tokens de diseño centralizados en `src/core/theme/tokens.ts`.
- **TypeScript**: Tipado estricto para mayor seguridad y mantenibilidad.

## 🎨 Estilo Visual
- **Colores**: Inspirados en la tradición gaucha (cueros, tierras, acentos dorados).
- **Tipografía**: 'Rye' para títulos (estilo tradicional) e 'Inter' para legibilidad en el cuerpo.
- **Logo**: Inspirado en una boina gaucha.

## 🧉 Secciones
- **Hero**: Introducción con impacto visual.
- **La Experiencia**: Detalle de las actividades (Asado, Cabalgata, Historia).
- **Contenido VIP**: Información sobre el servicio de fotografía y Dron.
- **Reserva**: Tarifas y llamada a la acción.
