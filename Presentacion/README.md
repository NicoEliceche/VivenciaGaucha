# Presentación para socios

Documento web independiente para presentar la propuesta de alianza de **Día de Gaucho - Experiencia en Magdalena**.

## Uso

Abrir `index.html` en un navegador moderno. La presentación permite:

- Navegar por scroll, menú lateral o teclas `↑`, `↓`, `Page Up` y `Page Down`.
- Usar el botón **Exportar PDF** para imprimir o guardar una versión PDF.
- Consultarla desde celular; el diseño es responsive.
- Abrirla sin conexión: imágenes y tipografías están incluidas localmente.

`Presentacion-Dia-de-Gaucho.pdf` es una exportación lista para revisar o compartir. Volver a
generarla desde el botón **Exportar PDF** después de modificar el HTML.

## Antes de circular

Completar junto al socio operativo:

- Nombre formal y datos del Centro Gaucho asociado.
- Capacidad mínima/máxima y calendario disponible.
- Itinerario, responsables y protocolos definitivos.
- Costos, precio final, reparto económico y comisiones.
- Seguros, permisos y alternativas por clima.
