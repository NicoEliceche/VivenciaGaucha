# Presentación vertical para Centro Gaucho

Documento independiente en formato **A4 vertical**, diseñado para exportar como PDF sin cortes automáticos de contenido.

## Uso

- Abrir `index.html` en Chrome o Edge.
- Revisar las 5 páginas verticales.
- Usar el botón **Exportar PDF**.
- En el diálogo de impresión seleccionar:
  - Destino: Guardar como PDF.
  - Papel: A4.
  - Orientación: Vertical.
  - Márgenes: Ninguno.
  - Gráficos de fondo: Activados.

Cada sección `.page` mide exactamente `210mm × 297mm`, tiene `break-after: page` y evita cortes internos.
